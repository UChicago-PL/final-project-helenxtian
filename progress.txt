Implemented:
1. Read-only tensor core
- File: src/HTensor/Types.hs
- Created a simple `Tensor` type backed by `ForeignPtr`
- Added `mkTensorRO` to validate shape when constructing a tensor
- Added `indexRO` so reads are bounds-checked
- The interface is read-only right now (no write function)

2. Header parser
- File: src/HTensor/Header.hs
- Added a parser for the 128-byte header
- Currently extracts rows and cols, dtype tag

3. Evaluation
- File: test/Main.hs
- small test suite that checks both the header parser and the read-only tensor API
    - `parseHeader valid header`
        - builds correct 128-byte header
        - checks parsed result matches expected rows/cols/dtype
    - `parseHeader short header`
        - passes too few bytes
        - confirms the parser fails instead of accepting invalid input
    - `mkTensorRO invalid shape`
        - checks that a zero dimension returns `InvalidShape`
    - `indexRO in/out bounds`
        - checks that a valid index returns the correct value
        - checks that an invalid index returns an `IndexOutOfBounds` error
- `cabal test all` shows all tests passing
